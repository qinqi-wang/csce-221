How to run the Program:
(Uses putty as the compiler)

1. Type in 
	g++ -std=c++11 rand-file.cpp
2. Press the number of the option you wish to select 
   to create an input file

	For example, if I wanted to have a file of 100 random integers, 
	I would type in 1, press enter, and type in 1 and enter again.
3. After creating the input file, type in
	make all
   To compile all code for the sort program
4. Follow assignment 2 when it comes to running the program, but also
   input the option for if you want the code in random, ascending, or
   descending order.
